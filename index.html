资源抓取任务创建器

填写你的 GitHub 仓库信息与目标网址，点击“生成 Issue 链接”，跳转到 GitHub 的新建 Issue 页面完成提交。 提交后，仓库的 Actions 会在带有 fetch-request 标签时下载该网址的资源并提交到仓库的 resources/ 目录。

跳转到新建 Issue

使用说明

  1. 在上方填写 owner、repo 与目标网址。
  2. 点击“生成 Issue 链接”，再点击右侧“跳转到新建 Issue”。
  3. 在 GitHub 的 Issue 页面中，手动为该 Issue 添加标签：fetch-request，然后提交。
  4. 等待 Actions 执行完成，资源会被下载并提交到仓库的 resources/ 目录。
